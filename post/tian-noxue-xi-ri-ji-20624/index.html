#LeetCode 139单词拆分
解题思路：动态规划
*设定一个vector<bool> dp(s.size() + 1, false)存储结果
*设定set来存储字典
*找到字典中的最大长度和最小长度

    for(int j = max(0, i - maxLength); j < i; ++j) {
        if (dp[j] && m.find(s.substr(j, i -j) != m.end()) {
            dp[i] = true;
            break;
        }
    }
    return dp[s.size()];
